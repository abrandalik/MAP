Xbox Kinect 3D scanner

I had the opportunity to try out a quite expensive 3D scanner and concluded that, for around €1000, it’s not that great.

So, I decided to look for a cheaper option.

I could build one like FabScan or MakerScanner, then I realized that I own an Xbox Kinect for the Xbox 360, which could surely be used as a 3D scanner—and it can!

So what do we need?

Kinect for Xbox 360 and Kinect AC USB adapter (just look on aliexpress)
or
Kinect for Windows

A windows PC

How to do it

First lets install required software

Skanect software (no longer officially supported.) Official download, Skanect Pro torrent Kinect for Windows SDK v1.6 (make sure you don’t have any other version installed)

Now simply connect your Kinect and start the Skanect software. If the Kinect is not detected, try a different version of the SDK.

Afterwards you can export the scan as in a 3D model format.
